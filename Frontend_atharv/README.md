# InsightPoint AI

InsightPoint is a modern, AI-powered news analysis dashboard. It allows users to upload documents or paste text to perform advanced analysis like sentiment detection, keyword extraction, and more.

## ‚ú® Features

- **Document Analysis**: Upload PDF, DOCX, or TXT files for instant processing.
- **Sentiment Tracking**: Visualize sentiment trends and distributions.
- **AI Insights**: Automatically extract keywords and summarize long articles.
- **Interactive Dashboard**: Modern, responsive UI built with React and Tailwind CSS.

## üöÄ Getting Started

### Prerequisites

Ensure you have **Node.js** (v18 or higher) installed on your system.

### Installation

1. Clone or download this repository.
2. Open your terminal in the project folder.
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

To start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:5173`.

### Building for Production

To create a production-ready build:
```bash
npm run build
```
